## Usage
This project is developed using Visual Studio (C++). Please ensure the following libraries are configured in your environment before building:
* OpenCV
* Eigen3
* OpenMP

Clone the repository and open the project solution (`.sln`) to build the executable:
```
git clone [https://github.com/5777777/thesis.git](https://github.com/5777777/thesis.git)
```

## Testing enviroment
